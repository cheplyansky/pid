# PID Controller Simulator

Симулятор PID-регулятора, использующий сгенерированный C-код для тестирования работы контроллера на модели системы первого порядка с обратной связью.

## Назначение

- Выполняет симуляцию PID-регулятора с заданными параметрами (P=3, I=2, Ts=0.01)
- Моделирует поведение системы с обратной связью
- Сохраняет результаты в CSV файл
- Строит график переходного процесса

## Файлы проекта

- `test_pid.c` — основной код симулятора
- `model_generated.h` / `model_generated.c` — сгенерированный код PID-регулятора
- `plot.py` — скрипт для визуализации

## Быстрый старт

### 1. Компиляция

```bash
gcc -o simulation test_pid.c model_generated.c -lm
```

### 2. Запуск симуляции
```bash
./simulation
```
После выполнения:
- В консоли появится таблица переходного процесса
- Данные сохранятся в results/simulation_data.csv

#### Параметры симуляции
Параметр	Значение
Target	1.0
P gain	3.0
I gain	2.0
Sample time (Ts)	0.01
Time constant	1.0
Tolerance	0.001
